## 📱 What trierarch does

trierarch brings an Arch Linux desktop with KDE Plasma to Android in a pocket-sized setup. It runs with Wayland and does not need Termux or Termux:X11. You get a Linux desktop you can carry in your pocket and use when you need a full desktop environment on the move.

It is made for users who want:
- A real Linux desktop on Android
- KDE Plasma with a clean interface
- Wayland support
- A setup that does not rely on Termux
- A personal desktop you can keep with you

## ✅ What you need

Before you start, make sure you have:

- An Android device with 64-bit support
- Enough free storage for the app and Linux files
- A stable internet connection
- Permission to install apps from the source you use
- A keyboard and mouse if you want easier desktop use

For best results, use:
- Android 10 or newer
- At least 4 GB of RAM
- 8 GB or more free storage
- A device with good CPU support for Android apps
